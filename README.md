# A10city Academy

Learn about digital authenticity, content provenance, and building trust in the AI age.

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) v18 or higher (v20 recommended)
- npm (comes with Node.js)

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/a10city/academy.git
   cd academy
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

### Running Locally

Start the development server:

```bash
npm run dev
```
